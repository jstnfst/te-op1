File format

FVER
4 bytes
AIFC version constant 0xA2805140
COMM
64 bytes
Mono, 16-bit, 22050 Hz, sowt (little-endian PCM)
APPL
1028 bytes
"op-1" signature + compact JSON + space padding. All knob values are 16-bit integers 0–32767.
SSND
varies
Raw PCM sample data. Silence (all zeros) produces a valid but soundless patch.

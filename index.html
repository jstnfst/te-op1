OP-1 FIELD // parameter reference fw 1.5.7

The OP-1 Field stores synth patches as AIFF-C files. Each file embeds a 1028-byte JSON blob describing the synth engine, two ADSR envelopes, one FX chain, and one LFO — all as 16-bit integers (0–32767) in fixed 8-element arrays. This page maps every known array index to a named parameter for firmware 1.5.7. See the GitHub repo for tools and raw-value scale notes, or the display-notes.md for observed raw↔display mappings per parameter.

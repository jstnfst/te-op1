OP-1 FIELD // knob layout
HOME LAYOUT MAPPINGS PATCH

Each OP-1 Field patch embeds a JSON blob in an AIFF-C container. The blob holds the synth engine, one envelope (ADSR for most synths; dyna_env for dbox), one FX chain, and one LFO - all as 16-bit integers (0–32767) in fixed 8-element arrays. This page maps every known array index to a named parameter. See Value Mappings for raw-to-display value mappings and display-notes.md for full scale documentation.
